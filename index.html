About Reel Ratings

Abstract The goal of this program is to allow a user to enter a movie title and try to predict what the rating of that movie woudl be.

Aspects

  • TheMovieDb API -movie parameters (actors, directors, budget etc.)
  • IMDb dataset from AWS -ratings from previous movies
  • Data Algorithms (weighting actors vs. directors vs. budgetâ€¦)
  • Query data about movie title or movie id to get parameters
  • Query previous ratings of rating parameters
  • Storing data to be accessed and used from different parts of the program
  • Statistical analysis on ratings (Data algorithms, linear recursion)
  • An interface/web app to take input and display results

Parameters in Use/Planned
  • Average movie rating
  • Actor movie ratings
  • Directors
  • Producers
  • Studio
  • Writers
  • Genre
  • Budget
  • Cinematographer
  • Music Composer
  • More to Come...
